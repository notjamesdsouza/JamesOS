A professional portfolio website showcasing expertise in e-commerce marketing, marketplace optimization, and digital growth strategies.

## 🎯 About the Project

JamesOS is a personal portfolio website for James Dsouza, a Marketplace Growth & E-commerce Marketing professional. The site demonstrates:

- **Marketplace Experience**: Work across 15+ platforms including Amazon, Flipkart, Myntra, and quick commerce platforms
- **Core Skills**: Marketplace SEO, product content strategy, catalog optimization, analytics, and AI-powered automation

- **Impact Metrics**: 3000+ SKUs managed, 6-8% sales growth, experience across 15+ marketplaces

## ✨ Key Features

- **Responsive Design**: Mobile-first approach with breakpoints for all devices
- **Scroll Animations**: Reveal animations using `.reveal` class
- **Counter Animations**: Dynamic number counters with data attributes
- **Copy-to-Clipboard**: Interactive contact buttons for email/phone
- **Dynamic Configuration**: External `config.js` for centralized link management
- **Performance Optimized**: Modular CSS structure for efficient styling
- **SEO Ready**: Meta tags, semantic HTML, structured data
