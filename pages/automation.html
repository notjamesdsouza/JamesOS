Automating Product Image Link Management Using Excel VBA

Designed a two-step Excel VBA automation that extracts and rearranges Dropbox image links, reducing a repetitive manual catalog process from over an hour to just 5–10 minutes while improving accuracy and consistency.

BUSINESS CHALLENGE

The Problem

Every product image uploaded for marketplace listings required individual Dropbox links. Since each SKU could contain multiple images, copying links one by one was repetitive, slow, and prone to human error. Whenever branding or product images were updated, the entire process had to be repeated.

  • Individual image links copied manually
  • Repetitive copy-paste work
  • Difficult to manage hundreds of product images
  • High possibility of incorrect image sequence
  • Time-consuming catalog updates

OLD WORKFLOW

Manual Process Before Automation

Product image updates required multiple manual steps across Google Drive, Dropbox, and Excel. Each product image link had to be copied individually, making the process slow and prone to human error.
01

Image Quality Check

Open the product collection folder in Google Drive and verify spelling, image alignment, arrows, and product features before approval.

02

Download Approved Images

Download the finalized image folder after completing the quality check and branding review.

03

Upload to Dropbox

Upload the complete folder to Dropbox to generate shareable image links for marketplace listings.

04

Copy Individual Links

Open every image separately, copy its Dropbox link, and repeat the process for all product images.

05

Paste into Excel

Paste every image URL into the correct Excel column while maintaining the required image sequence manually.

06

Final Verification

Review all links again to ensure the correct image order, reduce errors, and avoid incorrect marketplace uploads.

NEW WORKFLOW

Automated Process Using Excel VBA

Instead of copying every Dropbox image link manually, I built an Excel VBA solution that extracts and arranges image links automatically. The process reduced repetitive work and completed image link preparation in just 5–10 minutes.
01

Image Quality Check

Verify all product images for spelling, alignment, arrows, and product features before uploading.

02

Upload Images to Dropbox

Upload the approved image folder to Dropbox while ensuring every image follows the required naming convention.

03

Copy Folder Contents

Open the Dropbox folder in Opera browser, select all image files, copy them, and paste the data directly into Excel in one operation.

04

Run VBA Link Extractor

Execute the first VBA macro to extract valid Dropbox hyperlinks from the copied data automatically.

05

Run VBA Link Organizer

Execute the second VBA macro to arrange image links into the required sequence for marketplace uploads.

06

Ready for Upload

The final Excel sheet is generated with correctly ordered image links, reducing manual effort and completing the process in approximately 5–10 minutes.

HOW THE VBA WORKS

Automation Logic

The automation consists of two VBA macros that work together to extract Dropbox image hyperlinks and arrange them into the required sequence for marketplace uploads.
01

Paste Dropbox Data

Copy all image entries from the Dropbox folder and paste them into the Excel worksheet in a single operation.

02

Read Cell Data

The VBA macro scans each populated row and identifies the embedded Dropbox hyperlinks attached to the pasted files.

03

Extract Hyperlinks

The first macro extracts the actual Dropbox URLs and writes them into dedicated Excel cells, eliminating manual copy-and-paste work.

04

Identify Image Sequence

Based on the image filenames, the VBA recognizes the required order such as Main Image, S1, S2, S3 and other supporting images.

05

Arrange Image Links

The second VBA macro automatically rearranges all extracted links into the correct sequence required for marketplace uploads.

06

Generate Final Output

A clean, standardized Excel sheet is produced with correctly ordered image links, ready for bulk catalog uploads.
